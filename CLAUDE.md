# Musician's Friend — Dev Instructions

## Version bumping
Every commit MUST update the version string in index.html.
The version appears as: <span class="app-version">vX.X.X</span>
Always increment appropriately and include it in the same commit as the code change.

## Branch strategy
Always commit and push directly to main unless told otherwise.

## Commit message format
"vX.X.X description of change"
